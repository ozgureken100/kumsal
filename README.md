# Kumsal Yüzme Akademisi

Premium, mobil uyumlu ve SEO odaklı React + Vite web sitesi.

## Özellikler

- React + Vite
- Responsive tasarım
- PWA manifest ve service worker
- SEO meta ve Open Graph etiketleri
- Schema.org JSON-LD
- Yerel SEO sayfaları:
  - `/kizilay-yuzme-kursu`
  - `/batikent-yuzme-kursu`
  - `/sogutozu-yuzme-kursu`
- Lead form kayıt yapısı
- Sabit WhatsApp butonu

## Kurulum

```bash
npm install
npm run dev
```

## Üretim Build

```bash
npm run build
```

Build çıktısı `dist/` klasörüne üretilir.
